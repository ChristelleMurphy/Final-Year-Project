Project Aim

The aim of this project was to explore the ways in which you can display 3D art in an interactive and dynamic way.

Project Rational

I hoped to discover the features and possibilities that web-based graphics tools such as THREE.js provide. I hope to implement what I have learned to showcase my work in future projects
